## 2. Features Page (`/features`)

### SEO metadata
- **Title:** `قابلیت‌های نوبت — سیستم نوبت‌دهی کامل برای وردپرس`
- **Meta description:** تمام ۴۰+ قابلیت افزونه نوبت: تقویم جلالی، درگاه پرداخت ایرانی، پیامک، گوگل‌کلندر، گزارش‌گیری و بیشتر.
- **H1:** همه قابلیت‌های نوبت در یک نگاه

### دسته‌بندی‌ها (chip filter):
`همه` · `هسته اصلی` · `پرداخت` · `اطلاع‌رسانی` · `تقویم` · `یکپارچه‌سازی` · `توسعه‌دهنده`

### فهرست کامل قابلیت‌ها:

**هسته اصلی (Core):**
- مدیریت خدمات با قیمت، مدت زمان، و ظرفیت
- مدیریت کارکنان با ساعت کاری فردی
- تقویم جلالی یکپارچه (هم در ادمین، هم در فرم مشتری)
- محاسبه اسلات‌های خالی با در نظر گرفتن وقفه‌ها
- قوانین تعطیلات (ایام ملی، تعطیلات اختصاصی)
- قوانین زمان‌بندی: زودترین / دیرترین موعد رزرو
- نوبت‌های تکرارشونده (روزانه، هفتگی، ماهانه)
- وضعیت‌های نوبت: pending, confirmed, cancelled, pending_payment, pending_confirmation, completed
- مدیریت ظرفیت موازی (group bookings)

**پرداخت (Payments):**
- درگاه زرین‌پال (sandbox + production)
- درگاه آی‌دی‌پی (sandbox + production)
- درگاه نکست‌پی
- پرداخت حضوری (cash)
- استرداد سرورساید با ثبت لاگ
- تأیید پرداخت دوطرفه (callback + verify)

**اطلاع‌رسانی (Notifications):**
- SMS از طریق کاوه‌نگار، ملی‌پیامک، آی‌پی‌پنل
- ایمیل از طریق wp_mail با قالب HTML
- قالب‌های سفارشی با ۱۵+ متغیر پویا
- مدیریت رضایت: مشتری می‌تواند opt-out کند
- یادآوری خودکار N ساعت قبل از نوبت
- لاگ ارسال‌ها با وضعیت و زمان

**تقویم (Calendar):**
- نمای تقویم هفتگی / ماهانه در ادمین
- رنگ‌بندی بر اساس کارمند
- اتصال به Google Calendar (OAuth2)
- بلوکه شدن اسلات‌ها بر اساس freebusy گوگل
- خروجی ICS برای هر نوبت

**یکپارچه‌سازی (Integrations):**
- بلوک گوتنبرگ با تنظیمات service_id و staff_id
- ویجت المنتور
- شورت‌کد `[nowbat_booking]` و `[nowbat_appointments]`
- REST API عمومی (slots, booking) و ادمین (CRUD)
- فیلترهای وردپرس برای گسترش‌پذیری

**توسعه‌دهنده (Developer):**
- ۲۵+ اکشن و فیلتر سفارشی
- ابزارهای GDPR (anonymize, export)
- Rate limiting بر اساس IP
- هر اکشن داری signature کامل در مستندات
- نقش‌های سفارشی وردپرس با capability granular

## 3. How It Works Page (`/how-it-works`)

### SEO metadata
- **Title:** `چطور نوبت را نصب و راه‌اندازی کنیم — راهنمای گام‌به‌گام`
- **Meta description:** راهنمای کامل نصب افزونه نوبت روی وردپرس: از دانلود تا دریافت اولین رزرو در ۵ گام ساده.
- **H1:** از نصب تا اولین نوبت — ۵ گام

### ساختار ۵ گام:

| گام | عنوان | خلاصه محتوا |
|-----|-------|-----------|
| ۱ | **دانلود و نصب افزونه** | دانلود از مخزن وردپرس یا آپلود zip از طریق Plugins > Add New. تصویر صفحه آپلود. |

| ۳ | **افزودن خدمات و کارکنان** | از منوی Nowbat > Services یک خدمت با قیمت و مدت زمان اضافه کنید. از Nowbat > Staff یک کارمند با ساعت کاری اضافه کنید و خدمت را به او اختصاص دهید. |

| ۵ | **اولین نوبت می‌رسد** | مشتری فرم را تکمیل می‌کند، پرداخت می‌کند، پیامک تأیید دریافت می‌کند. شما در ادمین اعلان جدید می‌بینید. |

### 5.1 Getting Started

#### `/docs/getting-started/installation`
- **H1:** نصب افزونه نوبت
- پیش‌نیازها: وردپرس ۶.۰+، PHP 8.1+، MySQL 5.7+
- روش ۱: نصب از مخزن وردپرس (WP Admin > Plugins > Add New > search "Nowbat")
- روش ۲: آپلود دستی zip
- روش ۳: WP-CLI (`wp plugin install nowbat --activate`)
- تأیید نصب: منوی "نوبت" در سایدبار ادمین
- عیب‌یابی رایج نصب

#### `/docs/getting-started/first-setup`
- **H1:** راه‌اندازی اولیه
- ویزارد اولیه گام‌به‌گام
- تنظیم اطلاعات کسب‌وکار (نام، آدرس، شماره)
- تنظیم منطقه زمانی به Asia/Tehran
- تعریف ساعات کاری پیش‌فرض
- ایجاد اولین خدمت
- ایجاد اولین کارمند
- تست فرم با Shortcode

#### `/docs/getting-started/first-booking`
- **H1:** اولین رزرو — end-to-end walkthrough
- مشتری فرم را باز می‌کند
- انتخاب خدمت → کارمند → تاریخ → ساعت
- ورود اطلاعات تماس
- پرداخت از طریق زرین‌پال sandbox
- دریافت پیامک تأیید
- مدیر در ادمین نوبت را می‌بیند
- تأیید یا رد دستی توسط مدیر

### 5.3 Shortcodes
