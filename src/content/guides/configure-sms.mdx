# پیکربندی پیامک با کاوه‌نگار

کاوه‌نگار یکی از پایدارترین سرویس‌های پیامک ایران است. در این راهنما از صفر تا اولین پیامک موفق پیش می‌رویم.

## پیش‌نیازها

- نوبت نصب و فعال است
- یک شماره تلفن ایرانی برای ثبت‌نام در کاوه‌نگار دارید

---

## مرحله ۱: ثبت‌نام و دریافت API Key

<Steps>

### رفتن به سایت کاوه‌نگار

آدرس [app.kavenegar.com](https://app.kavenegar.com) را باز کنید و روی **ثبت‌نام** کلیک کنید.

### تکمیل ثبت‌نام

نام، ایمیل، و شماره موبایل را وارد کنید. کد تأیید SMS را دریافت و وارد کنید.

### پیدا کردن API Key

پس از ورود، از منوی **تنظیمات ← API Key** بخش «API Access Token» را باز کنید. API Key را کپی کنید.

<Callout type="info">
API Key کاوه‌نگار یک رشته طولانی است مثل `3D4C6B7E9F...`. آن را در جای امنی نگه دارید.
</Callout>

### خط ارسال

اگر خط اختصاصی دارید، شماره آن را یادداشت کنید. اگر ندارید، از خط اشتراکی استفاده می‌کنید (برای شروع کافی است).

</Steps>

---

## مرحله ۲: پیکربندی در نوبت

<Steps>

### رفتن به تنظیمات پیامک

در پنل وردپرس، به **پرونوبت ← تنظیمات ← پیامک** بروید.

### انتخاب کاوه‌نگار

از dropdown «ارائه‌دهنده پیامک» مقدار **Kavenegar** را انتخاب کنید.

### وارد کردن API Key

API Key کپی‌شده را در فیلد **API Key** وارد کنید.

### وارد کردن خط ارسال

شماره خط را در فیلد **خط ارسال** وارد کنید. اگر خط اشتراکی دارید، این فیلد را خالی بگذارید.

### ذخیره تنظیمات

**ذخیره** را بزنید.

</Steps>

---

## مرحله ۳: ارسال پیامک آزمایشی

<Steps>

### باز کردن بخش آزمایش

در همان صفحه تنظیمات پیامک، بخش **ارسال پیامک آزمایشی** را پیدا کنید.

### وارد کردن شماره آزمایشی

شماره موبایل خودتان را وارد کنید (فرمت `09XXXXXXXXX`).

### ارسال

دکمه **ارسال** را بزنید. در عرض چند ثانیه باید پیامک «نوبت - پیام آزمایشی» دریافت کنید.

</Steps>

---

## مرحله ۴: بررسی رویدادهای پیامک

در بخش **رویدادهای پیامک**، می‌توانید هر رویداد را جداگانه فعال یا غیرفعال کنید:

| رویداد | وضعیت پیشنهادی |
|---|---|
| تأیید نوبت | ✓ فعال |
| لغو نوبت | ✓ فعال |
| یادآور ۲۴ ساعته | ✓ فعال |
| تغییر زمان نوبت | ✓ فعال |

## عیب‌یابی

| مشکل | راه‌حل |
|---|---|
| «API Key نامعتبر» | کپی‌پیست کنید، فاصله اضافی نداشته باشد |
| «اعتبار ناکافی» | از پنل کاوه‌نگار حساب را شارژ کنید |
| پیامک دریافت نمی‌شود | فیلتر opt-out مشتری را بررسی کنید |
| پیامک ارسال می‌شود ولی لاگ خطا دارد | سقف روزانه `sms_daily_cap` را بالاتر ببرید |
