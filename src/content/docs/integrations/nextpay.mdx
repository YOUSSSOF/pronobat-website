# یکپارچه‌سازی نکست‌پی

## پیش‌نیازها

- حساب کاربری در [nextpay.org](https://nextpay.org)
- API Key از پنل «تنظیمات → API»

## پیکربندی در نوبت

مسیر: **پرونوبت ← تنظیمات ← پرداخت ← نکست‌پی**

| فیلد | توضیح |
|---|---|
| API Key | کلید API از پنل نکست‌پی |

## استرداد دستی

<Callout type="warning">
نکست‌پی از **استرداد خودکار** پشتیبانی نمی‌کند. هنگامی که درخواست استرداد ثبت شود، فیلتر `pronobat_refund_manual` فعال می‌شود و مدیر باید استرداد را از پنل نکست‌پی انجام دهد.
</Callout>

فرآیند استرداد دستی:

1. پرونوبت ← نوبت‌ها → جزئیات نوبت موردنظر را باز کنید
2. شماره مرجع (`ref_id`) را یادداشت کنید
3. وارد پنل نکست‌پی شوید
4. تراکنش را با شماره مرجع پیدا کنید
5. از پنل نکست‌پی استرداد را انجام دهید
6. وضعیت نوبت را به‌صورت دستی به `refunded` تغییر دهید

## اطلاع‌رسانی از استرداد دستی

```php
add_filter( 'pronobat_refund_manual', function( $is_manual, $appt_id, $payment ) {
    if ( $payment['gateway'] === 'nextpay' ) {
        // ارسال ایمیل به مدیر با اطلاعات تراکنش
        wp_mail(
            get_option('admin_email'),
            'استرداد دستی مورد نیاز است',
            "شماره مرجع: {$payment['ref_id']}\nمبلغ: {$payment['amount']} ریال"
        );
    }
    return $is_manual;
}, 10, 3 );
```
