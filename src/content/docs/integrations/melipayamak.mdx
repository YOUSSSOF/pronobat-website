# یکپارچه‌سازی ملی‌پیامک

## پیش‌نیازها

- حساب کاربری در [melipayamak.com](https://melipayamak.com)
- خط اختصاصی یا اشتراکی برای ارسال

## حالت‌های احراز هویت

ملی‌پیامک دو روش احراز هویت دارد:

### روش ۱: نام‌کاربری / رمز عبور

مسیر: **پرونوبت ← تنظیمات ← پیامک ← ملی‌پیامک**

| فیلد | توضیح |
|---|---|
| نام‌کاربری | نام‌کاربری پنل ملی‌پیامک |
| رمز عبور | رمز عبور پنل |
| خط ارسال | شماره خطی که پیامک از آن ارسال می‌شود |

### روش ۲: توکن

| فیلد | توضیح |
|---|---|
| توکن | توکن API از پنل «تنظیمات → API» |
| خط ارسال | شماره خطی که پیامک از آن ارسال می‌شود |

<Callout type="info">
استفاده از توکن نسبت به نام‌کاربری/رمز عبور امن‌تر است. در صورت امکان از این روش استفاده کنید.
</Callout>

## آزمایش ارسال

در صفحه تنظیمات پیامک، دکمه **ارسال پیامک آزمایشی** موجود است.

## خطاهای رایج

| کد | معنی |
|---|---|
| `0` | خطای نامشخص |
| `1` | موفق |
| `-1` | نام‌کاربری یا رمز اشتباه |
| `-2` | اعتبار ناکافی |
| `-3` | خط ارسال نامعتبر |
| `-4` | شماره مقصد نامعتبر |
