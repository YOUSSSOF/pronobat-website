# یکپارچه‌سازی آیدی‌پی

## پیش‌نیازها

- حساب کاربری در [idpay.ir](https://idpay.ir)
- API Key از منوی «تنظیمات → خدمات و API»

## پیکربندی در نوبت

مسیر: **پرونوبت ← تنظیمات ← پرداخت ← آیدی‌پی**

| فیلد | توضیح |
|---|---|
| API Key | کلید API از پنل آیدی‌پی |
| حالت | `sandbox` یا `production` |

## حالت Sandbox

در حالت Sandbox، API Key می‌تواند `6ba97b4bf5965b2735df7ab3d56ba137` باشد (مقدار تست آیدی‌پی).

## استرداد خودکار

آیدی‌پی از استرداد خودکار پشتیبانی می‌کند. برای استرداد وجه:

1. به **پرونوبت ← نوبت‌ها** بروید
2. نوبت پرداخت‌شده را باز کنید
3. دکمه **استرداد وجه** را کلیک کنید

نوبت درخواست استرداد را به API آیدی‌پی ارسال می‌کند و نتیجه را در لاگ تراکنش ثبت می‌کند.
