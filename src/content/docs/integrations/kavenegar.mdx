# یکپارچه‌سازی کاوه‌نگار

## پیش‌نیازها

- حساب کاربری در [kavenegar.com](https://kavenegar.com)
- API Key از پنل «تنظیمات → API Key»

## پیکربندی در نوبت

مسیر: **پرونوبت ← تنظیمات ← پیامک ← کاوه‌نگار**

| فیلد | توضیح |
|---|---|
| API Key | کلید API از پنل کاوه‌نگار |
| خط ارسال | شماره خطی که پیامک از آن ارسال می‌شود |

## قالب OTP

اگر می‌خواهید از قالب‌های از پیش تأیید‌شده کاوه‌نگار استفاده کنید:

1. در پنل کاوه‌نگار، قالب‌های خود را از بخش **ارسال قالب** تنظیم کنید
2. نام قالب را در نوبت در فیلد «نام قالب» وارد کنید

متغیرهای قالب کاوه‌نگار به‌طور خودکار از توکن‌های پیام پر می‌شوند.

## آزمایش ارسال

در صفحه تنظیمات پیامک، دکمه **ارسال پیامک آزمایشی** وجود دارد که یک پیامک نمونه به شماره‌ای که وارد می‌کنید ارسال می‌کند.

## اعتبار کافی

<Callout type="warning">
قبل از راه‌اندازی در محیط تولید، مطمئن شوید که اعتبار حساب کاوه‌نگار کافی است. در صورت کمبود اعتبار، ارسال پیامک بی‌صدا شکست می‌خورد.
</Callout>

## خطاهای رایج

| کد | معنی | راه‌حل |
|---|---|---|
| `200` | موفق | — |
| `401` | API Key نامعتبر | کلید را بررسی کنید |
| `403` | اعتبار ناکافی | حساب را شارژ کنید |
| `404` | شماره نامعتبر | فرمت `09XXXXXXXXX` را بررسی کنید |
| `412` | پارامتر ناقص | خط ارسال را بررسی کنید |
