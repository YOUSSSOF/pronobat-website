# هوک‌ها (Actions & Filters)

## Actions

### `pronobat_appointment_created`

هنگامی که یک نوبت جدید ثبت می‌شود فراخوانی می‌شود (قبل از تأیید نهایی).

```php
add_action( 'pronobat_appointment_created', function( int $id, array $appt ) {
    // $id: شناسه نوبت
    // $appt: آرایه کامل داده‌های نوبت
    error_log( "نوبت جدید: #{$id}" );
}, 10, 2 );
```

---

### `pronobat_appointment_status_changed`

```php
add_action( 'pronobat_appointment_status_changed', function(
    int $id, string $new_status, string $old_status, array $appt
) {
    if ( $new_status === 'confirmed' && $old_status === 'pending' ) {
        // اطلاع‌رسانی به کارمند
    }
}, 10, 4 );
```

## Filters

### `pronobat_available_slots`

لیست بازه‌های خالی را پیش از ارسال به کلاینت فیلتر می‌کند.

```php
add_filter( 'pronobat_available_slots', function(
    array $slots, int $staff_id, string $date, int $service_id
) {
    // حذف بازه ۱۲:۰۰ تا ۱۳:۰۰ برای همه روزها
    return array_filter( $slots, fn($s) => $s['start'] !== '12:00' );
}, 10, 4 );
```

---

### `pronobat_payment_gateways`

درگاه‌های پرداخت موجود را تغییر می‌دهد. برای افزودن درگاه سفارشی استفاده کنید.

```php
add_filter( 'pronobat_payment_gateways', function( array $gateways ) {
    $gateways['my_gateway'] = MyCustomGateway::class;
    return $gateways;
} );
```

---

### `pronobat_sms_message`

متن پیامک را قبل از ارسال تغییر می‌دهد.

```php
add_filter( 'pronobat_sms_message', function( string $msg, string $event, array $ctx ) {
    return $msg . "\n\nبا تشکر از اعتماد شما";
}, 10, 3 );
```

---

### `pronobat_sms_before_send`

کنترل می‌کند آیا پیامک ارسال شود یا نه.

```php
add_filter( 'pronobat_sms_before_send', function(
    bool $should_send, string $phone, string $message, array $ctx
) {
    // بلاک کردن ارسال در ساعت‌های شب
    $hour = (int) current_time( 'H' );
    if ( $hour >= 23 || $hour < 7 ) return false;
    return $should_send;
}, 10, 4 );
```

---

### `pronobat_refund_manual`

مشخص می‌کند آیا استرداد باید به‌صورت دستی انجام شود.

```php
add_filter( 'pronobat_refund_manual', function(
    bool $is_manual, int $appt_id, array $payment
) {
    // اگر مبلغ بیشتر از ۵ میلیون تومان بود، استرداد را دستی کن
    if ( $payment['amount'] > 50_000_000 ) return true;
    return $is_manual;
}, 10, 3 );
```
