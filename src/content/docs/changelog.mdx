# تاریخچه تغییرات

## نسخه ۱.۰.۰

**اولین نسخه عمومی پرونوبت**

### قابلیت‌های جدید

**سیستم رزرو:**
- فرم رزرو چند مرحله‌ای با تقویم جلالی
- مدیریت سرویس‌ها، کارمندان، و ساعت کاری
- قوانین تعطیلات و بلاک‌های زمانی
- تأیید خودکار یا دستی نوبت‌ها
- نوبت‌های تکرارشونده

**پرداخت:**
- زرین‌پال (sandbox + production)
- آیدی‌پی (با استرداد خودکار)
- نکست‌پی (با استرداد دستی)
- تأیید server-to-server برای همه درگاه‌ها

**اعلان‌ها:**
- پیامک: کاوه‌نگار، ملی‌پیامک، آی‌پی‌پنل
- ایمیل: قالب HTML با پشتیبانی از تم سفارشی
- یادآور خودکار ۲۴ ساعت قبل از نوبت
- سقف روزانه پیامک

**یکپارچه‌سازی:**
- Google Calendar OAuth2 (حالت‌های freebusy و full)
- بلاک Gutenberg
- ویجت Elementor
- شورت‌کد با ویژگی‌های سفارشی

**مدیریت:**
- داشبورد با آمار نوبت‌ها و درآمد
- گزارش هفتگی ایمیلی
- گزارش CSV قابل دانلود
- مدیریت نقش‌ها و permission ها

**امنیت:**
- محدودیت نرخ روی endpoint های عمومی
- تأیید Nonce برای تمام عملیات مدیریتی
- پاک‌سازی کامل ورودی‌ها
- ابزارهای GDPR (ناشناس‌سازی، صادر کردن)

**توسعه‌پذیری:**
- ۷ action و ۵ filter مستند
- رابط `PaymentGatewayInterface` برای درگاه سفارشی
- REST API کامل با مستندات
