# تنظیمات زبان و منطقه

مسیر: **پرونوبت ← تنظیمات ← زبان و منطقه**

## زبان افزونه

نوبت به‌طور پیش‌فرض از زبان وردپرس پیروی می‌کند. اگر وردپرس روی `fa_IR` تنظیم شده باشد، رابط کاربری، پیامک‌ها، و ایمیل‌ها به فارسی ارسال می‌شوند.

## تقویم

| گزینه | توضیح |
|---|---|
| جلالی (شمسی) | پیش‌فرض برای `fa_IR` — تمام تاریخ‌ها به شمسی نمایش داده می‌شوند |
| میلادی | برای کاربران بین‌المللی |

تقویم جلالی از `morilog/jalali` استفاده می‌کند و تمام تبدیل‌ها در `src/Services/JalaliCalendar.php` انجام می‌شوند.

## اعداد فارسی

وقتی زبان `fa_IR` باشد، تمام اعداد رو‌به‌روی کاربر (تاریخ، ساعت، مبلغ) به اعداد فارسی (۰۱۲۳۴۵۶۷۸۹) تبدیل می‌شوند.

برای غیرفعال کردن این قابلیت:

```php
add_filter( 'pronobat_use_persian_digits', '__return_false' );
```

## RTL / LTR

- پنل مدیریت پرونوبت همیشه RTL است.
- فرم رزرو عمومی جهت را از `<html dir>` ارث می‌برد.
- اگر وردپرس زبان انگلیسی داشته باشد، افزونه به‌طور خودکار LTR می‌شود.

## فرمت تاریخ

فرمت پیش‌فرض: `jYYYY/jMM/jDD` (مثلاً ۱۴۰۳/۰۴/۱۵)

برای تغییر فرمت:

```php
add_filter( 'pronobat_date_format', function() {
    return 'jD jF jY'; // مثلاً: پنجشنبه تیر ۱۴۰۳
} );
```
