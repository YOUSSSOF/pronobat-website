# ساعت کاری

مسیر: **پرونوبت ← تنظیمات ← ساعت کاری**

ساعت کاری پیش‌فرض برای همه کارمندان اعمال می‌شود. هر کارمند می‌تواند ساعت کاری مجزا داشته باشد.

## ساعت کاری هفتگی

برای هر روز هفته می‌توانید:

- **فعال/غیرفعال** — روزهایی که کار می‌کنید فعال کنید
- **ساعت شروع** — مثال: ۰۹:۰۰
- **ساعت پایان** — مثال: ۱۸:۰۰
- **زمان استراحت** — بازه‌ای که رزرو جدید قبول نمی‌شود (مثال: ۱۲:۰۰ — ۱۳:۰۰)

```
شنبه     ✓  09:00 — 18:00  استراحت: 12:00-13:00
یکشنبه   ✓  09:00 — 18:00
دوشنبه   ✓  09:00 — 18:00
سه‌شنبه  ✓  09:00 — 18:00
چهارشنبه ✓  09:00 — 17:00
پنجشنبه  ✓  09:00 — 13:00
جمعه     ✗  (تعطیل)
```

## تعطیلات عمومی

در بخش **تعطیلات**، تاریخ‌هایی که کسب‌وکار تعطیل است را اضافه کنید. مشتری نمی‌تواند در این روزها رزرو کند.

- تاریخ به شمسی وارد می‌شود.
- می‌توانید عنوان تعطیل را هم وارد کنید (مثال: «نوروز»).

## بلوک زمانی دستی

برای تعطیلی‌های موقت بدون تأثیر بر ساعت کاری عمومی، از **پرونوبت ← تقویم ← افزودن بلوک** استفاده کنید. این بلوک‌ها فقط یک بازه مشخص را می‌بندند.

## ساعت کاری اختصاصی کارمند

از **پرونوبت ← کارمندان ← ویرایش** می‌توانید برای هر کارمند ساعت کاری مجزا تعریف کنید. ساعت کاری کارمند بر ساعت کاری عمومی کسب‌وکار ارجحیت دارد.

## تأثیر بر محاسبه بازه‌ها

`AvailabilityCalculator` ترتیب زیر را برای تعیین ساعت‌های خالی رعایت می‌کند:

1. بلوک‌های دستی کارمند
2. رویدادهای Google Calendar کارمند (اگر متصل باشد)
3. نوبت‌های ثبت‌شده موجود
4. ساعت کاری اختصاصی کارمند (اگر تعریف شده باشد)
5. ساعت کاری عمومی کسب‌وکار
6. تعطیلات ثبت‌شده

اگر نتیجه خالی باشد، آن روز به مشتری نشان داده نمی‌شود.
