# تنظیمات عمومی

مسیر: **پرونوبت ← تنظیمات ← عمومی**

## فیلدها

### نام کسب‌وکار

نمایش داده شده در پیامک‌ها، ایمیل‌ها، و فرم رزرو.

- **نوع:** رشته متنی
- **پیش‌فرض:** نام سایت وردپرس

### منطقه زمانی

منطقه زمانی برای محاسبه بازه‌های خالی و تبدیل تاریخ جلالی.

- **پیش‌فرض:** `Asia/Tehran`
- **نکته:** تغییر این مقدار حافظه پنهان بازه‌های خالی را پاک می‌کند.

<Callout type="warning">
اگر بعد از تنظیم نوبت‌ها این مقدار را تغییر دهید، ساعت‌های ذخیره‌شده در پایگاه داده همچنان UTC هستند و نمایش آن‌ها در منطقه زمانی جدید درست خواهد بود.
</Callout>

### واحد پول

- **پیش‌فرض:** تومان
- گزینه‌ها: تومان، ریال

### فرمت تاریخ

- **پیش‌فرض:** `j F Y` (مثال: ۱ خرداد ۱۴۰۵)
- از توکن‌های PHP date استفاده می‌کند.

### حداقل مهلت رزرو

حداقل فاصله زمانی (دقیقه) بین زمان رزرو و شروع نوبت.

- **پیش‌فرض:** ۶۰
- مثال: اگر ۱۲۰ تنظیم شود، مشتری نمی‌تواند نوبت کمتر از ۲ ساعت دیگر بگیرد.

### حداکثر پیش‌رزرو

حداکثر تعداد روزی که مشتری می‌تواند از پیش رزرو کند.

- **پیش‌فرض:** ۳۰ روز

### تأیید خودکار

اگر فعال باشد، نوبت‌های جدید بلافاصله تأیید می‌شوند. اگر غیرفعال باشد، در وضعیت «در انتظار تأیید» باقی می‌مانند.

- **پیش‌فرض:** فعال

### حذف داده‌ها هنگام حذف افزونه

اگر فعال باشد، هنگام حذف افزونه از پیشخوان، تمام جداول و تنظیمات نوبت پاک می‌شوند.

- **پیش‌فرض:** غیرفعال

<Callout type="danger">
این گزینه را فقط هنگامی فعال کنید که می‌خواهید نوبت را برای همیشه حذف کنید. داده‌های حذف‌شده قابل بازگشت نیستند.
</Callout>
