# تنظیمات پیامک

مسیر: **پرونوبت ← تنظیمات ← پیامک**

## ارائه‌دهنده پیامک

یکی از سه ارائه‌دهنده را انتخاب کنید:

| ارائه‌دهنده | احراز هویت |
|---|---|
| کاوه‌نگار | API Key |
| ملی‌پیامک | نام‌کاربری / رمز عبور یا توکن |
| آی‌پی‌پنل | Access Key + خط ارسال |

## سقف روزانه ارسال

- **پیش‌فرض:** ۱۰۰ پیامک در روز
- این محدودیت از ارسال ناخواسته و هزینه اضافی جلوگیری می‌کند.
- شمارنده در نیمه‌شب (به وقت ایران) ریست می‌شود.

## رویدادهای پیامک

| رویداد | پیش‌فرض |
|---|---|
| نوبت تأییدشده | ✓ فعال |
| نوبت لغوشده | ✓ فعال |
| یادآور ۲۴ ساعته | ✓ فعال |
| تغییر زمان نوبت | ✓ فعال |
| درخواست استرداد تأیید شد | ✓ فعال |

## قالب پیام‌ها

قالب هر پیامک قابل ویرایش است. توکن‌های موجود:

| توکن | مقدار |
|---|---|
| `{customer_name}` | نام مشتری |
| `{service_name}` | نام خدمت |
| `{staff_name}` | نام کارمند |
| `{date}` | تاریخ جلالی |
| `{time}` | ساعت نوبت |
| `{business_name}` | نام کسب‌وکار |
| `{cancel_link}` | لینک لغو |

مثال پیش‌فرض تأییدیه:
```
{customer_name} عزیز، نوبت شما برای {service_name} در تاریخ {date} ساعت {time} با موفقیت ثبت شد.
— {business_name}
```

## فیلتر کردن پیام قبل از ارسال

```php
add_filter( 'pronobat_sms_message', function( string $msg, string $event, array $ctx ) {
    // اضافه کردن آدرس به پیامک تأییدیه
    if ( $event === 'appointment_created' ) {
        $msg .= "\nآدرس: خیابان ولیعصر، پلاک ۱۲";
    }
    return $msg;
}, 10, 3 );
```

## لغو opt-out

اگر مشتری گزینه «دریافت پیامک» را در پروفایل خود غیرفعال کرده باشد، `SmsService` پیامک ارسال نمی‌کند (حتی اگر فیلتر `pronobat_sms_before_send` مقدار `true` برگرداند).
