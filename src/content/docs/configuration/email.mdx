# تنظیمات ایمیل

مسیر: **پرونوبت ← تنظیمات ← ایمیل**

پرونوبت از سیستم WP Mail وردپرس برای ارسال ایمیل استفاده می‌کند. برای ارسال قابل‌اعتماد، پیشنهاد می‌شود از یک افزونه SMTP مانند **WP Mail SMTP** استفاده کنید.

## رویدادهای ایمیل مشتری

| رویداد | پیش‌فرض |
|---|---|
| تأیید نوبت | ✓ فعال |
| لغو نوبت | ✓ فعال |
| یادآور ۲۴ ساعته | ✓ فعال |
| تأیید استرداد | ✓ فعال |

## رویدادهای ایمیل مدیر

| رویداد | پیش‌فرض |
|---|---|
| نوبت جدید ثبت شد | ✓ فعال |
| نوبت لغوشده | ✓ فعال |
| پرداخت ناموفق | ✓ فعال |

## قالب ایمیل

ایمیل‌ها با قالب HTML ارسال می‌شوند. قالب از پوشه `wp-content/themes/{theme}/pronobat/emails/` بارگذاری می‌شود. اگر این پوشه وجود نداشت، قالب پیش‌فرض افزونه استفاده می‌شود.

برای سفارشی‌سازی قالب، فایل‌های زیر را از `wp-content/plugins/pronobat/templates/emails/` کپی کنید:

- `appointment-confirmed.php`
- `appointment-cancelled.php`
- `appointment-reminder.php`

## توکن‌های قالب

همان توکن‌های پیامک در قالب ایمیل هم کار می‌کنند (`{customer_name}`, `{date}` و غیره). علاوه بر آن:

| توکن | مقدار |
|---|---|
| `{appointment_id}` | شماره نوبت |
| `{payment_amount}` | مبلغ پرداخت‌شده |
| `{booking_link}` | لینک جزئیات نوبت |

## آزمایش ارسال

در صفحه تنظیمات ایمیل، دکمه **ارسال ایمیل آزمایشی** وجود دارد که یک ایمیل نمونه به آدرس ادمین می‌فرستد.
