# شورت‌کدها

## `[pronobat_booking]`

فرم رزرو کامل را نمایش می‌دهد. این شورت‌کد را در هر صفحه یا پست وردپرس قرار دهید.

### ویژگی‌ها

| ویژگی | پیش‌فرض | توضیح |
|---|---|---|
| `service_id` | — | نمایش مستقیم یک سرویس خاص (انتخاب سرویس از مرحله اول حذف می‌شود) |
| `staff_id` | — | نمایش مستقیم یک کارمند خاص |
| `theme` | `default` | `default` یا `minimal` |
| `show_price` | `true` | نمایش قیمت سرویس |
| `require_login` | `false` | الزام ورود کاربر برای رزرو |

### مثال‌ها

```
[pronobat_booking]
```

```
[pronobat_booking service_id="3" theme="minimal"]
```

```
[pronobat_booking staff_id="5" require_login="true"]
```

### در Gutenberg

بلاک **ProNobat Booking Form** در ویرایشگر Gutenberg در دسترس است. همان ویژگی‌های شورت‌کد از طریق پنل بلاک قابل تنظیم هستند.

### در Elementor

ویجت **ProNobat Booking** در دسته «Nowbat» موجود است.

## `[pronobat_appointments]`

لیست نوبت‌های کاربر وارد‌شده را نمایش می‌دهد.

```
[pronobat_appointments]
```

| ویژگی | پیش‌فرض | توضیح |
|---|---|---|
| `status` | `all` | فیلتر بر اساس وضعیت: `pending`, `confirmed`, `cancelled` |
| `limit` | `10` | تعداد نوبت‌ها در هر صفحه |

---

## `[pronobat_cancel]`

فرم لغو نوبت را نمایش می‌دهد. معمولاً در صفحه‌ای جداگانه قرار می‌گیرد که لینک لغو در ایمیل/پیامک به آن اشاره دارد.

```
[pronobat_cancel]
```

<Callout type="info">
توکن `{cancel_link}` در قالب پیامک و ایمیل به‌طور خودکار URL صفحه‌ای را تولید می‌کند که این شورت‌کد در آن قرار دارد. مطمئن شوید صفحه لغو ساخته شده و شورت‌کد در آن قرار دارد.
</Callout>
